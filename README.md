# Save Our Supper

A community food sharing web app built with React, TypeScript, Vite, Tailwind CSS, Firebase Firestore, Firebase Hosting, Supabase Auth, and Leaflet maps.

Live app: https://save-our-supper.web.app

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Current Features

- Firebase Firestore food listings
- Location-aware feed and Leaflet map markers
- Listing creation with postcode geocoding
- Claim flow with live Firestore updates
- Seed test data for realistic regional listings
- Live stock levels grouped from available Firestore posts
- Firebase Hosting deployment
